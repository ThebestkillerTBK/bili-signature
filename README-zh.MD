## ❓介绍

本程序可以根据自己的哔哩哔哩账号的粉丝数，自动更改您的个人简介。

## 🚀使用方法

运行 ``python bili-signature.py``

## ⚙配置

``SESSDATA`` 你的 SESSDATA.

``bili_jct`` 你的 bili_jct.

``freq`` 更新频率(秒).

``signature`` 你的个人简介，%d是粉丝数.

## 🚗运行

* 运行 ``pip install -r requirements.txt``.
* 编辑 ``config.json`` 来配置参数.
* 享受吧~
